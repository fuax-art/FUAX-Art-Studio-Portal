FUAX Art Generator Studio & Metadata Generator Guide

Overview

Welcome to the official guide for Forever Unfinished Artworx (FUAX) tools!
This guide will walk you through:

  • Generating unique NFT artwork with the FUAX Art Generator Studio (artgen.html)
  • Creating and editing metadata for your NFTs with the FUAX Metadata Generator (metagen.html)
  • Exporting your collection for use on platforms like OpenSea and beyond

1. Generating NFT Artwork with FUAX Art Generator Studio

  1. Open the FUAX Art Generator Studio:
    Open artgen.html in your browser.
  2. Set Up Your Collection:
    • Collection Name: Enter a name for your NFT collection.
    • AI Prompt: Describe the type of artwork you want (e.g., “A futuristic robot in a neon city”).
    • Number of Artworks: Choose how many unique artworks to generate (up to 1000).
  3. (Optional) Use Reference Images and Metadata:
    • Reference Image: Upload a PNG or JPG image to use as a visual base for the AI (optional).
    • Metadata (.json or .csv): Upload a file with trait data to guide the artwork generation (optional, but useful for trait consistency).
  4. Select Traits:
    Use the Traits Editor to pick or randomize traits for your collection (background, character, accessories, rarity, etc.).
  5. Generate Your Collection:
    Click Generate Collection. The app will use AI to create unique images based on your prompt, traits, and reference image. Preview the generated images in the right panel.
  6. Export Your Artwork:
    Use the Export Options to download your images as PNGs. You can also export trait data as JSON or CSV for use in the metadata generator.

2. Creating and Editing Metadata with FUAX Metadata Generator

  1. Open the Metadata Generator:
    Open metagen.html in your browser.
  2. Generate Metadata:
    Click Generate All Metadata to create 100 unique metadata entries for your NFTs. Each entry will have randomized traits (personality, digital affinity, energy level, etc.) with rarity weights.
  3. Edit Metadata (Optional):
    Click Edit Metadata to open the editor. Select an NFT number (1-100) to view and edit its details. Change the name, personality, digital affinity, energy level, special ability, or origin story as needed. Click Save Changes to update the entry.
  4. Export Metadata:
    The generated metadata appears in CSV format in the output box. Click Download CSV to save the metadata file. This file can be used for OpenSea Studio or other NFT platforms.

3. Using FUAX Art & Metadata Apps Together

  1. Generate Art: Use Art Generator Studio to create your NFT images. Download/export the images.
  2. Generate Metadata: Use the Metadata Generator to create and edit metadata for each NFT. Download the CSV file.
  3. Match Images and Metadata: Ensure the order/numbering of images matches the metadata IDs (e.g., image_001.png matches metadata row with ID 1). You can edit names and traits in the metadata generator to match the visual style of each image.
  4. Upload to NFT Platform: Use OpenSea Studio or your preferred platform. Upload your images and import the metadata CSV. Follow the platform’s instructions to publish your collection.
Tip: For trait consistency, upload a metadata CSV to the art generator before creating images. Edit metadata after image generation for best results.

FUAX Tips & Best Practices

  • Trait Consistency: If you want specific traits to appear in both the image and metadata, upload a metadata CSV to the art generator before creating images.
  • Editing: Use the metadata editor to fine-tune names and traits for each NFT after generating the initial set.
  • Backups: Always keep backups of your images and metadata files.
  • Testing: Preview your collection on OpenSea Studio before finalizing.

Troubleshooting FUAX Tools

  • File Upload Issues: Make sure you are using a modern browser and that file inputs are visible and clickable.
  • Image/Metadata Mismatch: Double-check that image filenames and metadata IDs correspond.
  • Export Problems: If downloads don’t work, try right-clicking the output and saving manually.

FUAX Workflow Summary

  • Use the Art Generator Studio to create your NFT images.
  • Use the Metadata Generator to create and edit metadata for your NFTs.
  • Export both, match them up, and upload to your NFT platform.
If you need more advanced automation (like auto-matching images and metadata), reach out for help!
