1. Metadata Generator - Start Here

Always create your metadata first. This establishes the traits and structure for your collection.

Step-by-Step:

  1. Choose Your Trait Set: Select Cyber/Tech, Monster, Human, or Alien
  2. Set Collection Size: Choose 1-25 entries (default: 13)
  3. Add Custom Name: Enter a prefix like "CyberPug" or use the random name generator
  4. Generate Metadata: Click "Generate Metadata" button
  5. Review in Table View: Toggle table view to see all traits clearly organized
  6. Download CSV: Save your metadata file
Rarity Scores: The generator automatically calculates rarity based on trait uniqueness. Higher scores = rarer combinations. Use this to identify your collection's most valuable pieces.

CSV Format:

Your downloaded CSV will include:

  • ID (numbered 1-25)
  • Name (with your custom prefix)
  • Description
  • 5 trait categories (varies by trait set)
  • Rarity Score

2. Using CSV with Art Generator

Upload your metadata CSV to generate images that match your traits.

Method 1: CSV-Driven Generation

  1. Open Art Generator Studio
  2. Upload Your CSV: Use "Collection Data" upload section
  3. Add Base Prompt: Enter style guidance (e.g., "digital art, neon colors, cyberpunk style")
  4. Select Additional Traits: Click trait options to add to the prompt
  5. Generate: The system combines CSV traits + your prompt + selected traits
How it works: For each CSV row, the generator creates a prompt like:
"[CSV Personality] [CSV Affinity] [CSV Energy] [Your Prompt] [Selected Traits]"

Method 2: Manual Trait Selection

  1. Don't upload CSV
  2. Select traits manually from the Traits Editor
  3. Enter your prompt
  4. Generate up to 5 images
Cost Control: Maximum 5 images per generation to manage Stability AI costs. Please purchase a package for larger collections and more features.

3. Preparing for OpenSea Upload

File Organization:

  1. Name Your Images: Match CSV IDs exactly
    • CSV Row 1 → 001.png
    • CSV Row 2 → 002.png
    • And so on...
  2. Keep Your CSV: You'll upload this to OpenSea
  3. Optional Videos: Name them 001.mp4, 002.mp4 to match

OpenSea Studio Upload Process:

  1. Create Collection: Set up your collection info on OpenSea
  2. Bulk Image Upload: Upload all your numbered PNG files in order
  3. Import CSV Metadata:
    • OpenSea has a "Import from CSV" option
    • Upload your metadata CSV
    • OpenSea matches by order (Image 1 = CSV Row 1)
  4. Add Videos Manually: OpenSea doesn't bulk-upload videos via CSV, add them individually to matching NFTs
  5. Set Pricing: Use rarity scores to guide pricing tiers
Pro Tip: Sort your CSV by Rarity Score before uploading. This helps you identify which images should have premium pricing.

5. Advanced Tips

Editing After Generation:

  • Open your CSV in Excel/Google Sheets
  • Edit names, descriptions, or traits as needed
  • Re-upload the modified CSV to OpenSea

Creating Trait Variations:

  • Generate multiple small batches with different trait sets
  • Combine CSVs manually in a spreadsheet
  • Renumber IDs sequentially
  • Create a mixed-trait collection

Rarity Strategy:

  • Top 20% rarity scores = "Legendary" tier pricing
  • Middle 30% = "Rare" tier
  • Bottom 50% = "Common" tier
  • Add "Rarity Tier" column to your CSV for OpenSea

Troubleshooting

CSV Not Loading in Art Generator:

  • Check file is actually .csv format (not .txt or .xlsx)
  • Open CSV in text editor - verify comma-separated format
  • Remove any extra blank rows at the end

OpenSea Metadata Mismatch:

  • Verify image numbering matches CSV row order EXACTLY
  • Check for duplicate IDs in CSV
  • Ensure no images were skipped in numbering

Images Not Generating:

  • Check browser console for error messages (F12)
  • Verify you have Stability AI credits remaining
  • Try simpler prompts if getting errors
  • Reduce generation count to 1 for testing
